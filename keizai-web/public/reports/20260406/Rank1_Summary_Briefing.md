# 経済情報：ホルムズ海峡の事実上封鎖と世界経済への影響

## エグゼクティブ・サマリー

2026年2月末の米国・イスラエルによるイラン攻撃と、それに伴うイラン革命防衛隊によるホルムズ海峡の事実上の封鎖を受け、世界経済は重大な供給ショックに直面している。世界の海上原油輸送量の約2割を占める同海峡の航行船舶数は急減し、原油価格の上昇やナフサ等の原材料の供給障害が顕在化している。

今後、中東からの供給は段階的な回復と代替ルートの活用により緩やかに正常化に向かうと想定されるが、各当事国の政治的思惑や軍事衝突の再発リスクにより、そのパスは極めて不確実である。本報告書では、原油価格の高騰が各国・地域のインフレ率を押し上げ、実質GDP成長率を下押しする定量的影響、および主要国・地域の経済対策と金融政策の動向について詳述する。

---

## 1. 主要テーマの詳細分析

### 1.1 ホルムズ海峡封鎖の現状と供給正常化へのパス
ホルムズ海峡は世界の石油関連取引の20%超、原油通航量で日量約2,000万バレル（うち約80%はアジア向け）を担う急所（チョークポイント）である。
*   **足元の状況:** 2026年3月29日時点の航行船舶数はわずか3隻まで急減。
*   **代替ルート:** サウジアラビアの「東西パイプライン」（輸送余力300～500万バレル/日）やUAEの「ハブシャン‐フジャイラ パイプライン」（同70万バレル/日）の活用が模索されているが、海峡の輸送能力を完全には代替できない。
*   **不確実性:** イランにとって海峡封鎖は外交上の「切り札」であり、正常化のペースは国内外の利害関係に左右される。また、停止した生産設備の再稼働にはタイムラグが生じる。

### 1.2 世界経済への価格影響と供給制約
OECDの推計に基づくと、原油価格が年間平均で25%上昇した場合、世界のCPIを約1.1%ポイント押し上げ、GDPを約0.2%ポイント押し下げる（金融影響を含めると▲0.4%ポイント）。
*   **供給障害の影響:** 価格面だけでなく、肥料（化学製品）などの関連商品の供給制約がアジアやアフリカの新興国に深刻な影響を及ぼすリスクがある。
*   **価格前提:** 2026年のWTI平均は1バレル70ドル台半ば～80ドル台半ばと想定される。

### 1.3 地域別経済への影響比較
エネルギー自給率や輸入依存度により、影響の濃淡が鮮明となっている。

## 2. 重要な引用と背景

> **「今後の中東地域からの原油等の供給は、各当事国にとっての政治・経済合理性などから、全体として早期に復旧が進むと想定する妥当性がなおもある。」**

*   **背景:** イランにとっても原油収入は国家財政の根幹であり、海峡の封鎖継続は中国を含む友好国への悪影響を通じて自国の孤立を深めるため、正常化へ向かうインセンティブが存在する。

> **「トランプ政権はイランとの早期の停戦等を模索するものとみられる。」**

*   **背景:** 2026年の中間選挙を控え、ガソリン価格の30%以上の急騰が政権支持率を低下させている。インフレ抑制が選挙戦略上の最優先課題となっている。

> **「植田日銀総裁は...原油価格の上昇が予想物価上昇率の引き上げを通じて基調的物価を押し上げる可能性と、景気悪化が基調的物価を押し下げる可能性の双方に言及。」**

*   **背景:** 日本銀行は、中東情勢によるインフレ圧力と景気後退リスクの両面を見極める必要があり、慎重な政策運営を迫られている。

## 3. 株式市場の反応

中東情勢の悪化は、業種によって明暗を分けている。

*   **上昇業種:** 資源価格高騰を背景とする「鉱業」、運賃上昇を見込んだ「海運」、エネルギー関連の「石油・石炭製品」などが逆行高。
*   **下落業種:** コスト増が直撃する「空運」、「ゴム製品」、「機械」、「輸送用機械」などが大きく値を下げている。
*   **地域差:** エネルギー純輸入国である韓国、インドネシア、インド、日本などのアジア諸国で株価の下落が目立つ一方、産油国のサウジアラビアなどは上昇している。

## 4. アクションに向けたインサイト

### 4.1 マーカー原油の価格差（スプレッド）への注視
通常、ドバイ原油はWTIに対し＋5ドル程度の差で推移するが、足元では中東リスクの織り込みによりドバイ原油がWTIの2倍近くに達する局面があった。日本を含むアジア企業は、ドバイ原油のスポット価格に基づいたコスト算定を精緻化する必要がある。

### 4.2 サプライチェーンと代替調達の検討
*   **肥料・化学製品:** アジア・アフリカ諸国では中東依存度が高い肥料の供給停滞が農業生産に影響する恐れがある。
*   **エネルギー源の多様化:** ASEAN諸国やインドでは、石炭の増産やロシア産原油の購入など、中東依存を低減させる動きが加速しており、これらの動向が国際政治に与える影響にも注意が必要である。

### 4.3 欧州における冬場のエネルギー供給リスク
欧州は天然ガスの約10%を中東（特にカタールからのLNG）に依存している。冬場に向けて在庫率を高めることができない場合、経済活動そのものが直接的に制限されるリスクがあり、産業界はエネルギー消費の抑制策を準備すべきである。

### 4.4 通貨防衛と金利リスク
米国の金利高止まりと原油高による経常収支悪化が重なることで、アジア通貨には下落圧力がかかりやすい。一部の国では景気悪化局面であっても通貨防衛のための利上げを余儀なくされる可能性があり、金融環境の引き締まりに備える必要がある。
